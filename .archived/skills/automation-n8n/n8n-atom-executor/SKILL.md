# n8n-atom-executor

## Overview
This skill provides context and patterns for executing and orchestrating automated workflows via the `n8n-atom` ecosystem, interacting via the configured MCP inspector bridge.

## Capabilities
- Triggering configured N8N webhooks.
- Inspecting workflow executions via `mcp-inspector-atom8n`.
- Deploying and orchestrating automation logic within the local stack.

## When to Use
This skill must be invoked whenever handling workflow execution, building webhook integrations, or debugging flow paths inside the central n8n context.
