---

## How Skills Work

| Concept | Description |
|---|---|
| **Auto-Activation** | Skills trigger automatically when your request matches a skill's domain |
| **Manual Activation** | Say "use the X skill" to explicitly load one |
| **SKILL.md** | Each skill folder contains a `SKILL.md` with full instructions |
| **Progressive Disclosure** | Minimal context loaded first, details on demand |

## Finding Skills

All skills live in `.agents/skills/<skill-name>/SKILL.md`. To browse:

**PowerShell:**
```powershell
Get-ChildItem -Path ".agents\skills" -Directory | Measure-Object                                  # Count
Get-ChildItem -Path ".agents\skills" -Directory | Where-Object { $_.Name -like "*react*" }        # Search
```

**Bash / Zsh:**
```bash
ls -d .agents/skills/*/ | wc -l                  # Count
ls -d .agents/skills/*react*/                     # Search
```

**Cross-platform (Node.js):**
```bash
npx -y glob ".agents/skills/*/SKILL.md" | wc -l  # Count
npx -y glob ".agents/skills/*react*/SKILL.md"     # Search
```

Or just ask: *"Is there a skill for X?"*
